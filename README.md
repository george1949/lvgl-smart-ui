- **💻 软件环境**：VsCode、Ubuntu、LVGL、Mosquitto
- **🎛️ 硬件平台**：嵌入式 ARM 开发板 (如 S5P6818 等)、PAJ7620U2 手势识别传感器
## ✨ 核心技术与实现
      基于 POSIX 线程与互斥锁构建异步并发模型，实现底层 IIC 手势阻塞轮询与前端LVGL图形渲染的物理隔离，保障高频交互场景下的 UI 帧率稳定
      基于 PAJ7620U2 传感器数据流构建“语义过滤层”，结合 300ms 时间窗口防抖算法完成原始手势特征向界面状态机与设备二态控制逻辑的抗干扰映射
      运用面向对象思想构建 C 语言多设备抽象层，实现底层硬件状态与 LVGL 控件指针的内存级深度绑定，统一异构设备的管理与调度接口
      基于 MQTT 协议搭建云端发布/订阅通信链路，采用轻量化 JSON 报文与 Linux 后台子进程完成非阻塞异步指令下发，实现中控端与多受控设备间的毫秒级状态同步
